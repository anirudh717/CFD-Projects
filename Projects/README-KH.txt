This is a 2D - Kelvin-Helmhltz instability simulator

This is close to a 2D DNS as no assumptions on Turbulence models have been made.

A Pseudo-spectral combination has been utilized here for the computational purposes.

Sponge Boundary Conditions have been added to counter the effect of periodicity of the spectral scheme.

GPU-Acceleration with CUDA has been used here.

Vortical structures have been visualized as a result
